## SX1276 Register Definitions

The file `sx1276Regs-Fsk.h` contains register definitions for the Semtech SX1276 transceiver chip.

**Source:** Based on Semtech SX1276 datasheet  
**Purpose:** Low-level register access for RF communication

## Standard Libraries

This project also uses standard Arduino and ESP32 libraries:

- **SPI** - Serial Peripheral Interface communication
- **Wire** - I2C communication library
- **WiFi** - ESP32 WiFi functionality
- **HTTPUpdateServer** - OTA firmware update support
- **ESP32 Core Libraries** - Base ESP32 functionality

These libraries are part of the ESP32 Arduino Core and are licensed under the LGPL-2.1 license.
